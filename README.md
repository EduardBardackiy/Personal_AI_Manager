# Personal AI Manager 🤖📅

**Personal AI Manager** — это интеллектуальный ассистент для управления временем, встречами и задачами.  
Бот создан в конструкторе Савви https://app.suvvy.ai/ в рамках интенсива по промпт инжинерингу проводимого университетом  "Zerocoder". Бот интегрирован с **Google Calendar** и помогает пользователю планировать день с учетом личного распорядка, уровня загрузки и приоритетов задач.

Проект реализован с использованием **no-code инструментов** и методов **prompt engineering**.

---

# Overview

Personal AI Manager — это не просто календарный бот.  
Он работает как **умный тайм-менеджер**, который помогает:

- управлять событиями в календаре
- избегать временных конфликтов
- учитывать личный распорядок дня
- находить свободные слоты
- планировать задачи
- оптимизировать расписание

Перед созданием или изменением событий бот анализирует:

1. **Google Calendar**
2. **личное расписание пользователя**
3. **временные ограничения**
4. **потенциальные конфликты**

Благодаря этому пользователь получает **более реалистичный и удобный план дня**.

# Key Features

## Calendar Management

Бот интегрирован с **Google Calendar** и умеет:

- создавать события
- изменять существующие события
- удалять события
- искать события по запросу пользователя
- проверять доступность времени
- предлагать свободные интервалы
- предупреждать о временных конфликтах
- запрашивать подтверждение перед изменениями

---

## Knowledge Base Integration

Бот использует файл **daily_routine.txt**, который хранится в базе знаний.

Файл содержит **постоянный распорядок пользователя по дням недели**.

Бот умеет:

- находить нужный день недели
- учитывать фиксированные блоки времени
- учитывать гибкие блоки
- проверять пересечения с событиями

Типы блоков:

**Blocked blocks**

Неприкосновенные временные интервалы, которые нельзя занимать.

**Flexible blocks**

Блоки, которые можно перенести только с согласия пользователя.

---

## Conflict Detection

Бот автоматически проверяет конфликты между:

- событиями Google Calendar
- личным расписанием
- новыми встречами

Он умеет:

- обнаруживать пересечение временных интервалов
- считать частичное пересечение конфликтом
- запрещать создание события в заблокированное время
- запрашивать разрешение для гибких блоков
- предлагать альтернативные слоты

---

## Smart Meeting Creation

Пользователь может создавать встречи в свободной форме.

Примеры:
Создай встречу завтра в 15:00
Запланируй звонок на понедельник
Назначь встречу на вторую половину дня


Бот извлекает из запроса:

- название события
- дату
- время начала
- продолжительность

Если данных недостаточно — бот задаёт уточняющие вопросы.

Стандартная длительность:

| Тип события | Продолжительность |
|--------------|------------------|
meeting | 60 minutes |
call | 30 minutes |

Перед созданием события бот:

1. проверяет daily_routine.txt  
2. определяет день недели  
3. проверяет blocked / flexible блоки  
4. проверяет Google Calendar  
5. предлагает альтернативы при конфликте  
6. запрашивает подтверждение  

---

## Smart Time Suggestions

Если пользователь не знает, когда назначить встречу, бот может предложить свободное время.

Он умеет:

- искать свободные интервалы
- учитывать личное расписание
- анализировать календарь
- предлагать до **3 удобных слотов**

Примеры запросов:

Когда можно созвониться?
Найди свободное время завтра
Предложи время для встречи


---

## Event Editing

Бот может редактировать события:

- менять дату
- менять время
- менять продолжительность
- проверять конфликты после изменений
- повторно проверять расписание

Перед сохранением изменений бот **всегда запрашивает подтверждение**.

---

## Event Deletion

Бот может удалять события.

Он умеет:

- находить нужное событие
- уточнять, если найдено несколько вариантов
- запрашивать подтверждение
- удалять событие после подтверждения

## Daily Schedule Overview

Бот может показать план на день.

Например:

Что у меня сегодня?
Покажи мой план на день
Какие встречи завтра?


Ответ включает:

- события из Google Calendar
- фиксированные блоки из daily_routine.txt
- краткий обзор дня

## Daily Briefing

Бот может делать **ежедневный обзор дня**.

Включает:

- встречи
- личные блоки
- важные события
- рекомендации по загрузке

Он может:

- предложить добавить задачи
- предупредить о перегруженности
- обратить внимание на неудобные интервалы

## AI Task Planner

Бот умеет планировать задачи.

Пользователь может отправить список задач, например:

Задачи на сегодня:
— подготовить презентацию
— ответить на письма
— провести исследование


Бот:

1. анализирует задачи  
2. оценивает продолжительность  
3. находит свободные слоты  
4. предлагает готовый план  

Стандартная логика длительности:

| Тип задачи | Время |
|-------------|------|
small task | 30 min |
normal task | 60 min |
complex task | 90–120 min |

После подтверждения задачи добавляются в календарь.

---

## Energy-Based Planning

Бот распределяет задачи по уровню энергии:

| Время | Тип задач |
|------|-----------|
Morning | сложные и творческие задачи |
Day | встречи и обсуждения |
Evening | лёгкие задачи |

Это помогает делать расписание **более реалистичным и комфортным**.

---

## Schedule Optimization

Бот может анализировать структуру дня и давать рекомендации.

Например:

- день перегружен встречами
- расписание слишком фрагментировано

Он может предложить:

- перенести встречи

- используется для сложных задач

# Communication Style

Особенности:

# Safety Rules

Перед выполнением действий бот:

- проверяет daily_routine.txt
- проверяет Google Calendar
- анализирует конфликты
- запрашивает подтверждение пользователя

Это предотвращает:

- случайные пересечения событий
- нарушение личного расписания
- создание событий без согласия пользователя
